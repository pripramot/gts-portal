## 1. ภาพรวมระบบ

### 1.1 ACESO คืออะไร?

**ACESO Forensic Portal** เป็นระบบเว็บแอปพลิเคชันสำหรับเจ้าหน้าที่ ป.ป.ส. ใช้ในการ:

- 📍 **ติดตามตำแหน่ง** - ค้นหาตำแหน่งจากหมายเลขโทรศัพท์
- 📁 **จัดการเอกสาร** - อัปโหลดและจัดเก็บไฟล์คดี
- 📊 **รายงานคดี** - สร้างและดูรายงานคดีต่างๆ
- 🖥️ **Remote Access** - เข้าถึงระบบ ACESO จากระยะไกล

### 1.2 คุณสมบัติหลัก

| คุณสมบัติ | รายละเอียด |
|-----------|------------|
| ติดตามตำแหน่ง | ค้นหาพิกัดจากเบอร์โทรศัพท์ แสดงบน Google Maps |
| บันทึกข้อมูล | จัดเก็บข้อมูลการติดตามลง Firebase |
| อัปโหลดไฟล์ | รองรับไฟล์ทุกประเภท เก็บใน Google Drive |
| รายงาน | ส่งออกรายงานเป็น PDF |
| รองรับมือถือ | ใช้งานได้ทั้ง Android และ iOS |

### 1.3 ความปลอดภัย

- ✅ การเชื่อมต่อ HTTPS เข้ารหัสทั้งหมด
- ✅ บันทึก Log การเข้าใช้งานทุกครั้ง
- ✅ ต้องระบุหน่วยงานและเลขที่หนังสือ
- ✅ ข้อมูลจัดเก็บบน Firebase (Google Cloud)

## 3. การติดตั้งและเข้าถึง

### 3.1 การเข้าถึงผ่านเว็บ (ทุกอุปกรณ์)

ACESO เป็น Web Application ไม่ต้องติดตั้ง เข้าใช้งานผ่าน URL:

```
https://rungroj-carrent.web.app
```

หรือ

```
https://rungroj-carrent.firebaseapp.com
```

### 3.2 การเพิ่มไอคอนบน Android (Add to Home Screen)

**ขั้นตอน:**

1. เปิด **Chrome** บนมือถือ Android
2. ไปที่ URL: `https://rungroj-carrent.web.app`
3. แตะ **เมนู ⋮** (มุมขวาบน)
4. เลือก **"เพิ่มไปยังหน้าจอหลัก"** หรือ **"Add to Home screen"**
5. ตั้งชื่อ: **ACESO**
6. แตะ **"เพิ่ม"**

```
┌─────────────────────────────────────┐
│  Chrome Menu                    ⋮   │
├─────────────────────────────────────┤
│  🔖 บุ๊กมาร์ก                        │
│  ⬇️  ดาวน์โหลด                       │
│  📱 เพิ่มไปยังหน้าจอหลัก  ← คลิกนี้   │
│  🖥️  เวอร์ชันเดสก์ท็อป                │
│  ⚙️  การตั้งค่า                      │
└─────────────────────────────────────┘
```

**ผลลัพธ์:** ไอคอน ACESO จะปรากฏบนหน้าจอหลัก สามารถเปิดใช้งานได้เหมือนแอปทั่วไป

### 3.3 การเพิ่มไอคอนบน iOS (Add to Home Screen)

**ขั้นตอน:**

1. เปิด **Safari** บน iPhone/iPad
2. ไปที่ URL: `https://rungroj-carrent.web.app`
3. แตะ **ไอคอนแชร์** ⬆️ (ตรงกลางด้านล่าง)
4. เลื่อนลงและแตะ **"เพิ่มไปยังหน้าจอโฮม"**
5. ตั้งชื่อ: **ACESO**
6. แตะ **"เพิ่ม"** (มุมขวาบน)

```
┌─────────────────────────────────────┐
│  Safari Share Menu             ⬆️   │
├─────────────────────────────────────┤
│  📋 คัดลอก                          │
│  📖 เพิ่มไปยัง Reading List          │
│  🔖 เพิ่มบุ๊กมาร์ก                    │
│  📱 เพิ่มไปยังหน้าจอโฮม  ← คลิกนี้    │
│  ✉️  ส่งทางเมล                      │
└─────────────────────────────────────┘
```

> ⚠️ **หมายเหตุ:** ต้องใช้ Safari เท่านั้น Chrome บน iOS ไม่รองรับ Add to Home Screen

### 7.3 วิธีอัปโหลด

3. ลากมาวางในกรอบ "ลากไฟล์มาวางที่นี่"

**วิธีที่ 2: คลิกเลือก**

1. คลิกที่กรอบ "ลากไฟล์มาวางที่นี่"
2. เลือกไฟล์จากอุปกรณ์
3. กด "เปิด" หรือ "Open"

3. เลือก "รูปภาพและไฟล์ในแคช"
4. กด "ล้างข้อมูล"

## 10. การแก้ไขปัญหาเบื้องต้น

### 10.1 ปัญหาการเชื่อมต่อ

**อาการ:** หน้าจอแสดง "ไม่สามารถเชื่อมต่อได้" หรือโหลดช้ามาก

**วิธีแก้ไข:**
1. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
   - ลองเปิดเว็บไซต์อื่น เช่น google.com
2. รีเฟรชหน้าเว็บ
   - กดปุ่มรีเฟรช หรือดึงหน้าจอลงแล้วปล่อย
3. ลองเปลี่ยนจาก WiFi เป็น 4G หรือกลับกัน
4. รีสตาร์ทเบราว์เซอร์
5. รีสตาร์ทอุปกรณ์

### 10.2 ปัญหาการเข้าสู่ระบบ

**อาการ:** กดเข้าสู่ระบบแล้วไม่มีอะไรเกิดขึ้น

**วิธีแก้ไข:**
1. ตรวจสอบว่ากรอกข้อมูลครบทุกช่อง
2. ตรวจสอบว่าไม่มีช่องว่างหน้า/หลังข้อความ
3. ล้างแคชเบราว์เซอร์
4. ลองใช้เบราว์เซอร์อื่น

### 10.3 ปัญหาแผนที่ไม่แสดง

**อาการ:** กรอบแผนที่ว่างเปล่าหรือแสดง Error

**วิธีแก้ไข:**
1. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
2. รอสักครู่แล้วลองใหม่ (อาจเป็นปัญหา Google Maps)
3. ล้างแคชเบราว์เซอร์
4. ลองใช้ VPN หากอยู่ในเครือข่ายที่จำกัด

### 10.4 ปัญหาการอัปโหลดไฟล์

**อาการ:** อัปโหลดไม่สำเร็จ หรือค้างอยู่

**วิธีแก้ไข:**
1. ตรวจสอบขนาดไฟล์ (ต้องไม่เกิน 100 MB)
2. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
3. ลองลดจำนวนไฟล์ที่อัปโหลดพร้อมกัน
4. ลองใช้ WiFi แทน 4G (เสถียรกว่า)

### 10.5 ปัญหาหน้าจอค้าง

**อาการ:** หน้าจอไม่ตอบสนอง กดอะไรก็ไม่ได้

**วิธีแก้ไข:**
1. รอสักครู่ (อาจกำลังประมวลผล)
2. รีเฟรชหน้าเว็บ
3. ปิดแท็บแล้วเปิดใหม่
4. ปิดเบราว์เซอร์แล้วเปิดใหม่
5. รีสตาร์ทอุปกรณ์

## 11. คำถามที่พบบ่อย

### Q1: ACESO ใช้งานได้ฟรีหรือไม่?
**A:** ใช่ ACESO เป็นระบบภายในสำหรับเจ้าหน้าที่ ไม่มีค่าใช้จ่าย

### Q2: ข้อมูลที่บันทึกจะถูกเก็บไว้นานแค่ไหน?
**A:** ข้อมูลจะถูกเก็บไว้บน Firebase ตลอดไป จนกว่าจะถูกลบโดยผู้ดูแลระบบ

### Q3: สามารถใช้งานแบบ Offline ได้หรือไม่?
**A:** ไม่ได้ ACESO ต้องการการเชื่อมต่ออินเทอร์เน็ตตลอดเวลา

### Q4: ใครสามารถเห็นข้อมูลที่ฉันค้นหา?
**A:** เฉพาะผู้ดูแลระบบเท่านั้นที่สามารถเข้าถึงข้อมูลทั้งหมดได้

### Q5: รองรับภาษาอื่นนอกจากภาษาไทยหรือไม่?
**A:** ปัจจุบันรองรับเฉพาะภาษาไทย

### Q6: สามารถใช้งานบนคอมพิวเตอร์ได้หรือไม่?
**A:** ได้ เปิดเว็บไซต์ผ่านเบราว์เซอร์บนคอมพิวเตอร์ได้เลย

### Q7: ทำไมต้องกรอกเลขที่หนังสือ?
**A:** เพื่อบันทึกการเข้าใช้งานตามระเบียบ และตรวจสอบย้อนหลังได้

### Q8: ข้อมูลตำแหน่งแม่นยำแค่ไหน?
**A:** ความแม่นยำขึ้นอยู่กับหลายปัจจัย โดยทั่วไปอยู่ที่ 10-50 เมตร

### Q9: สามารถติดตามหลายเบอร์พร้อมกันได้หรือไม่?
**A:** ปัจจุบันติดตามได้ทีละ 1 เบอร์ ต้องค้นหาใหม่ทุกครั้ง

### Q10: ลืมข้อมูลเข้าระบบทำอย่างไร?
**A:** ไม่มีรหัสผ่าน เพียงกรอกชื่อ หน่วยงาน และเลขที่หนังสือใหม่
