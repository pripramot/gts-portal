🚀 เริ่มต้นอย่างรวดเร็ว

เริ่มใช้งาน ACESO ได้ภายใน 5 นาที

ขั้นตอนที่ 1: เข้าถึงระบบ

เปิดเบราว์เซอร์และไปที่:

https://rungroj-carrent.web.app

หรือสแกน QR Code ด้านล่าง

ขั้นตอนที่ 2: เข้าสู่ระบบ

กรอกข้อมูลต่อไปนี้:

  1. ชื่อ-นามสกุล - ยศและชื่อของคุณ
  2. หน่วยงาน - หน่วยงานที่สังกัด
  3. เลขที่หนังสือ - เลขที่หนังสืออนุมัติ

ขั้นตอนที่ 3: เลือกฟีเจอร์

จากหน้า Dashboard เลือกฟีเจอร์ที่ต้องการใช้งาน:

  • 📍 Location Tracker - ติดตามตำแหน่ง
  • 📁 File Upload - อัปโหลดไฟล์

📍 ติดตามตำแหน่ง

ค้นหาตำแหน่งจากหมายเลขโทรศัพท์

วิธีใช้งาน

1. เข้าสู่หน้า Location Tracker

จาก Dashboard กด "📍 Location Tracker" หรือเปิด URL:

/location-tracker.html

2. กรอกข้อมูล

ช่อง ตัวอย่าง
เบอร์โทรศัพท์ 066-442-3732
ชื่อคดี คดีค้นเมทแอมเฟตามีน
เลขที่หนังสือ ที่ 123/2569

3. ดูผลลัพธ์

ระบบจะแสดง:

  • พิกัด (Latitude, Longitude)
  • ที่อยู่โดยประมาณ
  • IMSI และเครือข่าย
  • แผนที่ Google Maps

⚡ MCP API Reference

Model Context Protocol API สำหรับนักพัฒนา

การติดตั้ง

npm install @modelcontextprotocol/sdk @modelcontextprotocol/ext-apps

Tools ที่มี

Tool คำอธิบาย
track_location ติดตามตำแหน่งจากเบอร์โทร
verify_user ยืนยันตัวตนผู้ใช้
create_case สร้างคดีใหม่
get_location_history ดึงประวัติการติดตาม
generate_report สร้างรายงานคดี

ตัวอย่างการใช้งาน

{ "tool": "track_location", "arguments": { "phone": "066-442-3732", "caseName": "คดีทดสอบ", "docNumber": "ที่ 123/2569" } }

❓ คำถามที่พบบ่อย

คำตอบสำหรับคำถามที่พบบ่อย

Q: ACESO ใช้งานได้ฟรีหรือไม่?

A: ใช่ ACESO เป็นระบบภายในสำหรับเจ้าหน้าที่ ไม่มีค่าใช้จ่าย

Q: สามารถใช้งานแบบ Offline ได้หรือไม่?

A: ไม่ได้ ACESO ต้องการการเชื่อมต่ออินเทอร์เน็ตตลอดเวลา

Q: รองรับเบราว์เซอร์อะไรบ้าง?

A: Chrome, Safari, Firefox, Edge เวอร์ชันล่าสุด

Q: ข้อมูลตำแหน่งแม่นยำแค่ไหน?

A: ความแม่นยำอยู่ที่ประมาณ 10-50 เมตร ขึ้นอยู่กับหลายปัจจัย

🔧 แก้ไขปัญหา

วิธีแก้ไขปัญหาที่พบบ่อย

หน้าจอขาวหรือโหลดไม่ขึ้น

  1. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
  2. ล้างแคชเบราว์เซอร์
  3. ลองใช้เบราว์เซอร์อื่น
  4. รีสตาร์ทอุปกรณ์

แผนที่ไม่แสดง

  1. ตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
  2. รอสักครู่แล้วลองใหม่
  3. ล้างแคชเบราว์เซอร์

ไม่สามารถอัปโหลดไฟล์ได้

  1. ตรวจสอบขนาดไฟล์ (ไม่เกิน 100 MB)
  2. ลองลดจำนวนไฟล์ที่อัปโหลดพร้อมกัน
  3. ใช้ WiFi แทน 4G

กรอกข้อมูล 3 ช่อง:

  • ชื่อ-นามสกุล - พร้อมยศ
  • หน่วยงาน - หน่วยงานที่สังกัด

📁 อัปโหลดไฟล์

  3. รอจนเสร็จสมบูรณ์

ขีดจำกัด

  • ขนาดไฟล์สูงสุด: 100 MB ต่อไฟล์

🚀 วิธีใช้งาน

ทดสอบในเครื่อง

เปิดไฟล์ index.html ด้วย Browser หรือรันคำสั่ง:

npm install
npm run serve

Deploy ขึ้น Firebase Hosting (แนะนำ)

npm install -g firebase-tools
firebase login
firebase init hosting
firebase deploy --only hosting

จะได้ URL: https://rungroj-carrent.web.app

Deploy ด้วย GitHub Pages (ฟรี)

  1. สร้าง GitHub Repository
  2. Push โค้ดขึ้น GitHub
  3. ไปที่ Settings → Pages
  4. เลือก Branch: main → Folder: / (root)
  5. Save — จะได้ URL ทันที

Deploy ด้วย Netlify (ฟรี ง่ายที่สุด)

  1. ไปที่ netlify.com แล้วสมัคร (ฟรี)
  2. ลากโฟลเดอร์โปรเจกต์วางในหน้าเว็บ
  3. เสร็จ! ได้ URL ทันที
